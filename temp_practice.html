Question 1: Swap two numbers without using a third variable

#include <stdio.h>

int main() {
    int a = 10, b = 20;
    
    printf("Before swap: a = %d, b = %d\n", a, b);
    
    // Swap without third variable
    a = a + b;  // a becomes 30
    b = a - b;  // b becomes 10 (original a)
    a = a - b;  // a becomes 20 (original b)
    
    printf("After swap: a = %d, b = %d\n", a, b);
    
    return 0;
}

Answer:

How it works:

This program swaps two numbers without using a temporary third variable by using arithmetic operations. Let's understand step by step with initial values a=10 and b=20:

  1. Step 1: a = a + b → a becomes 30 (10 + 20). Now a stores the sum of both numbers.
  2. Step 2: b = a - b → b becomes 10 (30 - 20). We subtract the original b from the sum to get the original value of a.
  3. Step 3: a = a - b → a becomes 20 (30 - 10). We subtract the new b (which is original a) from the sum to get the original value of b.

Output:

Before swap: a = 10, b = 20
After swap: a = 20, b = 10

Alternative Method: You can also use XOR bitwise operator: a = a ^ b; b = a ^ b; a = a ^ b;

Question 2: Check if a number is an Armstrong number

#include <stdio.h>
#include <math.h>

int main() {
    int num, originalNum, remainder, result = 0, n = 0;
    
    printf("Enter an integer: ");
    scanf("%d", &num);
    
    originalNum = num;
    
    // Count number of digits
    while (originalNum != 0) {
        originalNum /= 10;
        ++n;
    }
    
    originalNum = num;
    
    // Calculate sum of nth power of each digit
    while (originalNum != 0) {
        remainder = originalNum % 10;
        result += pow(remainder, n);
        originalNum /= 10;
    }
    
    // Check if sum equals original number
    if (result == num)
        printf("%d is an Armstrong number.\n", num);
    else
        printf("%d is not an Armstrong number.\n", num);
    
    return 0;
}

Answer:

What is an Armstrong Number?

An Armstrong number (also called narcissistic number) is a number that is equal to the sum of its own digits each raised to the power of the number of digits.

Examples:

  • 153 is an Armstrong number: 153 = 1³ + 5³ + 3³ = 1 + 125 + 27 = 153 ✓
  • 9474 is an Armstrong number: 9474 = 9⁴ + 4⁴ + 7⁴ + 4⁴ = 6561 + 256 + 2401 + 256 = 9474 ✓
  • 123 is NOT an Armstrong number: 123 ≠ 1³ + 2³ + 3³ = 1 + 8 + 27 = 36 ✗

How the program works:

  1. Count digits: First loop counts how many digits are in the number (n)
  2. Extract each digit: Use modulo operator (%) to get the last digit: remainder = num % 10
  3. Calculate power: Raise each digit to the power of n using pow(remainder, n)
  4. Sum up: Add all the powered digits together
  5. Compare: If the sum equals the original number, it's an Armstrong number

Sample Output:

Enter an integer: 153
153 is an Armstrong number.

Question 3: Write a recursive function to find factorial

#include <stdio.h>

int factorial(int n) {
    // Base case: factorial of 0 or 1 is 1
    if (n == 0 || n == 1)
        return 1;
    
    // Recursive case: n! = n × (n-1)!
    return n * factorial(n - 1);
}

int main() {
    int num;
    
    printf("Enter a positive integer: ");
    scanf("%d", &num);
    
    if (num < 0) {
        printf("Factorial is not defined for negative numbers.\n");
    } else {
        printf("Factorial of %d = %d\n", num, factorial(num));
    }
    
    return 0;
}

Answer:

What is Factorial?

Factorial of a number n (written as n!) is the product of all positive integers less than or equal to n.

Formula: n! = n × (n-1) × (n-2) × ... × 2 × 1

Examples:

  • 5! = 5 × 4 × 3 × 2 × 1 = 120
  • 4! = 4 × 3 × 2 × 1 = 24
  • 3! = 3 × 2 × 1 = 6
  • 0! = 1 (by definition)

How Recursion Works:

Recursion is when a function calls itself. For factorial:

  1. Base Case: if (n == 0 || n == 1) return 1; - This stops the recursion
  2. Recursive Case: return n * factorial(n - 1); - Function calls itself with n-1

Example: factorial(5)

factorial(5) = 5 × factorial(4)
             = 5 × 4 × factorial(3)
             = 5 × 4 × 3 × factorial(2)
             = 5 × 4 × 3 × 2 × factorial(1)
             = 5 × 4 × 3 × 2 × 1
             = 120

Sample Output:

Enter a positive integer: 5
Factorial of 5 = 120

Question 4: Program for matrix multiplication (3×3)

#include <stdio.h>

int main() {
    int a[3][3], b[3][3], result[3][3];
    int i, j, k;
    
    // Initialize result matrix to 0
    for(i = 0; i < 3; i++) {
        for(j = 0; j < 3; j++) {
            result[i][j] = 0;
        }
    }
    
    // Input first matrix
    printf("Enter elements of first 3x3 matrix:\n");
    for(i = 0; i < 3; i++) {
        for(j = 0; j < 3; j++) {
            printf("a[%d][%d]: ", i, j);
            scanf("%d", &a[i][j]);
        }
    }
    
    // Input second matrix
    printf("\nEnter elements of second 3x3 matrix:\n");
    for(i = 0; i < 3; i++) {
        for(j = 0; j < 3; j++) {
            printf("b[%d][%d]: ", i, j);
            scanf("%d", &b[i][j]);
        }
    }
    
    // Matrix multiplication
    for(i = 0; i < 3; i++) {
        for(j = 0; j < 3; j++) {
            for(k = 0; k < 3; k++) {
                result[i][j] += a[i][k] * b[k][j];
            }
        }
    }
    
    // Display result
    printf("\nResultant Matrix:\n");
    for(i = 0; i < 3; i++) {
        for(j = 0; j < 3; j++) {
            printf("%d\t", result[i][j]);
        }
        printf("\n");
    }
    
    return 0;
}

Answer:

Matrix Multiplication Rule:

For two matrices A (m×n) and B (n×p), the result C will be (m×p). Each element C[i][j] is calculated as:

C[i][j] = A[i][0]×B[0][j] + A[i][1]×B[1][j] + ... + A[i][n-1]×B[n-1][j]

How the program works:

  1. Initialize: Create three 3×3 matrices (a, b, result) and set result to all zeros
  2. Input: Read elements for matrix a and matrix b from user
  3. Multiply: Use three nested loops:
    • i loop: iterates through rows of first matrix
    • j loop: iterates through columns of second matrix
    • k loop: performs the actual multiplication and addition
  4. Formula: result[i][j] += a[i][k] * b[k][j] - multiply row of A with column of B
  5. Display: Print the resulting matrix

Example:

Matrix A:        Matrix B:        Result:
1  2  3          7  8  9          30  36  42
4  5  6    ×    10 11 12    =    66  81  96
7  8  9          13 14 15         102 126 150

Calculation for result[0][0]:

result[0][0] = (1×7) + (2×10) + (3×13) = 7 + 20 + 39 = 66

Question 5: Check if a string is palindrome (without using string.h)

#include <stdio.h>

int main() {
    char str[100];
    int i, length = 0, flag = 1;
    
    printf("Enter a string: ");
    scanf("%s", str);
    
    // Calculate length manually
    while(str[length] != '\0') {
        length++;
    }
    
    // Check palindrome
    for(i = 0; i < length/2; i++) {
        if(str[i] != str[length - i - 1]) {
            flag = 0;
            break;
        }
    }
    
    if(flag == 1)
        printf("%s is a palindrome.\n", str);
    else
        printf("%s is not a palindrome.\n", str);
    
    return 0;
}

Answer:

What is a Palindrome?

A palindrome is a word, phrase, or sequence that reads the same backward as forward.

Examples:

  • Palindromes: madam, racecar, noon, level, radar
  • Not Palindromes: hello, world, programming

How the program works:

  1. Input string: Read the string from user
  2. Calculate length: Use a while loop to count characters until null terminator '\0'
  3. Compare characters: Compare first character with last, second with second-last, and so on
  4. Loop runs: Only until middle of string (length/2) to avoid redundant comparisons
  5. Flag variable: Initially set to 1 (true). If any mismatch found, set to 0 (false) and break

Example with "madam":

Length = 5
Comparisons:
  str[0] vs str[4]: 'm' == 'm' ✓
  str[1] vs str[3]: 'a' == 'a' ✓
  str[2] is middle, no need to compare
Result: Palindrome!

Example with "hello":

Length = 5
Comparisons:
  str[0] vs str[4]: 'h' != 'o' ✗
Result: Not a palindrome!

Sample Output:

Enter a string: radar
radar is a palindrome.

Question 6: File handling - Read numbers and write even/odd to separate files

#include <stdio.h>

int main() {
    FILE *inputFile, *evenFile, *oddFile;
    int num;
    
    // Open input file for reading
    inputFile = fopen("numbers.txt", "r");
    if(inputFile == NULL) {
        printf("Error opening input file!\n");
        return 1;
    }
    
    // Open output files for writing
    evenFile = fopen("even.txt", "w");
    oddFile = fopen("odd.txt", "w");
    
    if(evenFile == NULL || oddFile == NULL) {
        printf("Error creating output files!\n");
        return 1;
    }
    
    // Read numbers and separate even/odd
    while(fscanf(inputFile, "%d", &num) != EOF) {
        if(num % 2 == 0)
            fprintf(evenFile, "%d\n", num);
        else
            fprintf(oddFile, "%d\n", num);
    }
    
    // Close all files
    fclose(inputFile);
    fclose(evenFile);
    fclose(oddFile);
    
    printf("Numbers separated successfully!\n");
    printf("Even numbers written to even.txt\n");
    printf("Odd numbers written to odd.txt\n");
    
    return 0;
}

Answer:

File Handling in C:

File handling allows programs to store data permanently on disk. C provides functions like fopen(), fclose(), fprintf(), fscanf() for file operations.

How the program works:

  1. Open input file: fopen("numbers.txt", "r") - Opens file in read mode ("r")
  2. Check if file exists: If fopen returns NULL, file doesn't exist or can't be opened
  3. Create output files: fopen("even.txt", "w") - Opens/creates file in write mode ("w")
  4. Read numbers: fscanf(inputFile, "%d", &num) - Reads integers from file until EOF (End Of File)
  5. Check even/odd: num % 2 == 0 - If remainder is 0, number is even
  6. Write to files: fprintf(evenFile, "%d\n", &num) - Writes number to appropriate file
  7. Close files: Always close files using fclose() to save changes and free resources

File Modes:

  • "r" - Read mode (file must exist)
  • "w" - Write mode (creates new file or overwrites existing)
  • "a" - Append mode (adds to end of file)
  • "r+" - Read and write mode

Example:

numbers.txt:
12 15 8 23 40 17 6 9

After running program:

even.txt:        odd.txt:
12               15
8                23
40               17
6                9

Important Notes:

  • Always check if file opened successfully (check for NULL)
  • Always close files after use to prevent data loss
  • Use appropriate file modes based on operation

Question 7: Program using structure to store and display employee records

#include <stdio.h>
#include <string.h>

struct Employee {
    int id;
    char name[50];
    float salary;
    char department[30];
};

int main() {
    struct Employee emp[5];
    int i;
    
    // Input employee records
    printf("Enter details of 5 employees:\n\n");
    for(i = 0; i < 5; i++) {
        printf("Employee %d:\n", i+1);
        
        printf("  ID: ");
        scanf("%d", &emp[i].id);
        
        printf("  Name: ");
        scanf(" %[^\n]", emp[i].name);  // Read string with spaces
        
        printf("  Salary: ");
        scanf("%f", &emp[i].salary);
        
        printf("  Department: ");
        scanf(" %[^\n]", emp[i].department);
        
        printf("\n");
    }
    
    // Display all records
    printf("\n========== EMPLOYEE RECORDS ==========\n\n");
    for(i = 0; i < 5; i++) {
        printf("Employee %d:\n", i+1);
        printf("  ID: %d\n", emp[i].id);
        printf("  Name: %s\n", emp[i].name);
        printf("  Salary: %.2f\n", emp[i].salary);
        printf("  Department: %s\n", emp[i].department);
        printf("--------------------------------------\n");
    }
    
    // Find highest paid employee
    int maxIndex = 0;
    for(i = 1; i < 5; i++) {
        if(emp[i].salary > emp[maxIndex].salary) {
            maxIndex = i;
        }
    }
    
    printf("\nHighest Paid Employee:\n");
    printf("Name: %s\n", emp[maxIndex].name);
    printf("Salary: %.2f\n", emp[maxIndex].salary);
    
    return 0;
}

Answer:

What is a Structure?

A structure is a user-defined data type that allows you to group different types of variables under a single name. It's useful for representing real-world entities with multiple attributes.

Structure Syntax:

struct StructureName {
    datatype member1;
    datatype member2;
    ...
};

How the program works:

  1. Define structure: Create Employee structure with 4 members (id, name, salary, department)
  2. Declare array: struct Employee emp[5] - Array of 5 employee structures
  3. Input data: Loop through array and read data for each employee using emp[i].member notation
  4. Access members: Use dot operator (.) to access structure members: emp[i].salary
  5. Display records: Loop through array and print all employee details
  6. Find maximum: Compare salaries to find highest paid employee

Key Concepts:

  • Structure declaration: Defines the template
  • Structure variable: Creates actual storage
  • Member access: Use dot (.) operator
  • Array of structures: Multiple records of same type

  Department: IT

  ID: 101

  Department: IT

Question 8: Explain the difference between structure and union with example

Answer:

Structure vs Union - Key Differences:

Feature Structure Union
Keyword struct union
Memory Each member gets separate memory All members share same memory
Size Sum of all members' sizes Size of largest member
Access All members accessible anytime Only one member at a time
Value Storage All members can hold values simultaneously Only last assigned member holds value
Use Case When you need all data together When you need only one value at a time

Example Program:

#include <stdio.h>

// Structure example
struct Student {
    int roll;
    char grade;
    float marks;
};

// Union example
union Data {
    int i;
    char c;
    float f;
};

int main() {
    struct Student s1;
    union Data d1;
    
    // Structure - all members can hold values
    s1.roll = 101;
    s1.grade = 'A';
    s1.marks = 95.5;
    
    printf("=== STRUCTURE ===\n");
    printf("Roll: %d\n", s1.roll);      // 101
    printf("Grade: %c\n", s1.grade);    // A
    printf("Marks: %.2f\n", s1.marks);  // 95.50
    printf("Size: %lu bytes\n\n", sizeof(s1));  // 12 bytes (4+1+4+padding)
    
    // Union - only one member holds value at a time
    printf("=== UNION ===\n");
    
    d1.i = 10;
    printf("After d1.i = 10:\n");
    printf("  d1.i = %d\n", d1.i);     // 10 (correct)
    
    d1.c = 'A';
    printf("\nAfter d1.c = 'A':\n");
    printf("  d1.c = %c\n", d1.c);     // A (correct)
    printf("  d1.i = %d\n", d1.i);     // Garbage (overwritten)
    
    d1.f = 3.14;
    printf("\nAfter d1.f = 3.14:\n");
    printf("  d1.f = %.2f\n", d1.f);   // 3.14 (correct)
    printf("  d1.i = %d\n", d1.i);     // Garbage (overwritten)
    printf("  d1.c = %c\n", d1.c);     // Garbage (overwritten)
    
    printf("\nSize: %lu bytes\n", sizeof(d1));  // 4 bytes (size of float)
    
    return 0;
}

Memory Representation:

STRUCTURE:
┌─────────┬───────┬─────────┐
│  roll   │ grade │  marks  │
│ 4 bytes │1 byte │ 4 bytes │
└─────────┴───────┴─────────┘
Total: 12 bytes (with padding)

UNION:
┌─────────────┐
│   Shared    │  ← All members use this same space
│   4 bytes   │
└─────────────┘
Total: 4 bytes (largest member)

When to Use:

  • Use Structure: When you need to store employee record (id, name, salary all needed together)
  • Use Union: When you need to store either integer OR float OR char (only one type at a time), saves memory

Real-world Example:

Union is useful in embedded systems or when dealing with different data interpretations of the same memory location (like reading sensor data that could be int or float based on sensor type).
